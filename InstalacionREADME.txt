Para realizar la instalacion de Andromeda Desktop siga los siguientes pasos:

1. Abrir el archivo MenuVendedor.exe.config con el bloc de notas
2. Cambiar el connectionString, copiando la direccion donde se encuentra la base de datos "MicromercadoDB.mdf" 
   justo despues del "AttachDbFilename=" y antes del ";"
3. Guardar cambios
4. Salir de MenuVendedor.exe.config
5. Ejecutar MenuVendedor.exe
